# 📊 Sales & Feedback Dashboard

**Live App:** [Click here to view the Streamlit Dashboard]https://salesfeedbackproject-7hajuxy6civxbevfasjjra.streamlit.app/

### About This Project
This project is an interactive web application built with Streamlit and Python. It analyzes thousands of customer sales records and processes text feedback using Natural Language Processing (TextBlob) to determine customer sentiment.

### Technologies Used
* **Streamlit:** Web framework
* **Plotly:** Interactive charts
* **TextBlob:** Sentiment analysis
* **Pandas & NumPy:** Data manipulation
